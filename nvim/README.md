# myArchy 🐧

My custom Arch Linux setup managed with **GNU Stow**.

## 🛠️ Components
- **Editor**: Neovim
- **Shell**: (Fish)
- **WM/DE**: (Hyprland)

## 🚀 How to use
1. Clone this repo: `git clone git@github.com:Hemang-codes/myArchy.git ~/dotfiles`
2. Install GNU Stow: `sudo pacman -S stow`
3. Link configs: `cd ~/dotfiles && stow nvim`
